## Минимум слева
Ограничение времени - 1 с

Ограничение памяти - 256.0 Мб

### Ввод
стандартный ввод или input.txt
### Вывод
стандартный вывод или output.txt

Дан массив a, состоящий из n чисел.

Для каждой позиции i (1 ≤ i ≤ n) вычислите величину b<sub>i</sub> = min(a[1], a[2], …, a[i]).

### Формат ввода
Первая строка содержит единственное число n - количество чисел в массиве.

Вторая строка содержит n чисел a<sub>i</sub>, где a<sub>i</sub> - число на i-й позиции в массиве a. Числа разделены пробелами.

### Формат вывода
Выведите n, разделенных пробелами, чисел b <sub>i</sub>i.

### Примечание
Ограничения:
1 ≤ n ≤ 10<sup>5</sup>; 1 ≤ a<sub>i</sub>i ≤ 10<sup>5</sup> для всех 1≤i≤n.
