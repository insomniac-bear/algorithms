## Перепад цен
Ограничение времени - 1 с

Ограничение памяти - 256.0 Мб

### Ввод
стандартный ввод или input.txt
### Вывод
стандартный вывод или output.txt

Дан массив a, состоящий из n чисел. Необходимо найти две пары индексов i<sub>1</sub> < j<sub>1</sub> и i<sub>2</sub> < j<sub>2</sub> таких, что a<sub>i1</sub> - a<sub>j1</sub> минимально возможное и a<sub>i2</sub> - a<sub>j2</sub> максимально возможное.

### Формат ввода
Первая строка содержит единственное число n - количество чисел в массиве.

Вторая строка содержит n разделенных пробелом чисел a<sub>i</sub>, где a<sub>i</sub> - число на i-ой позиции в массиве a

### Формат вывода
Первая строка вывода должна содержать два разделенных пробелом числа i<sub>1</sub> и j<sub>1</sub>
. Если подходящих пар несколько, то нужно выбрать пару с минимальным значеним i<sub>1</sub>. Если пар с минимальным значением i<sub>1</sub> несколько, то нужно выбрать пару с минимальным значеним j<sub>1</sub>.

Вторая строка вывода должна содержать два разделенных пробелом числа i<sub>2</sub> и j<sub>2</sub>
. Если подходящих пар несколько, то нужно выбрать пару с минимальным значеним i<sub>2</sub>. Если пар с минимальным значением i<sub>2</sub> несколько, то нужно выбрать пару с минимальным значеним j<sub>2</sub>.

### Примечание
Ограничения:
* 1 ≤ n ≤ 10<sup>5</sup>
* 1 ≤ a<sub>i</sub> ≤ 10<sub>5</sub> для всех 1 ≤ i ≤ n
