## Поиск элемента в массиве
Ограничение времени - 1 с

Ограничение памяти - 256.0 Мб

### Ввод
стандартный ввод или input.txt
### Вывод
стандартный вывод или output.txt

Дан массив a, состоящий из n чисел, и q запросов. Каждый запрос состоит из одного целого числа x<sub>i</sub>. Найдите и выведите индекс p<sub>i</sub>, который является первым, самым левым, вхождением числа x<sub>i</sub> в массивa a. Если x<sub>i</sub> не встречается среди элементов массива, то p<sub>i</sub> = −1.

### Формат ввода
Первая строка содержит два разделенных пробелом числа n и q - количество чисел в массиве и количество запросов, соответственно.

Вторая строка содержит n чисел a<sub>i</sub>, где a<sub>i</sub>- число на i-й позиции в массиве a. Числа разделены пробелами.

Далее идут q строк, каждая строка содержит ровно одно число x<sub>i</sub> - очередной запрос.

### Формат вывода
Выведите q чисел p<sub>i</sub>i.

### Примечание
Ограничения:
* 1 ≤ n,q ≤ 10<sup>5</sup>;
* 1 ≤ a<sub>i</sub>≤ 10<sup>5</sup> для всех 1 ≤ i ≤ n;
* 1 ≤ x<sup>i</sup> ≤10<sup>5</sup> для всех 1 ≤ i ≤ q.
